## Objective
To detect fraudulent credit card transactions using a Random Forest model.

## Dataset
- Credit Card Fraud Dataset  
- Target column: `Class` (0 = Normal, 1 = Fraud)

## Tools Used
- Python  
- Pandas  
- Scikit-learn  

## Steps
1. Load the dataset  
2. Check class imbalance (fraud vs non-fraud)  
3. Split data using stratified sampling  
4. Train a baseline model  
5. Train Random Forest model  
6. Evaluate using precision, recall, and F1-score  
7. Save the trained model  

## Evaluation
- Precision  
- Recall  
- F1-score  
- Feature Importance  

## Outcome
Learned ensemble learning and how to handle imbalanced datasets using Random Forest.
